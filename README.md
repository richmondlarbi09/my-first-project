# My First Project - C++ Calculator

This is a simple calculator made with C++. It can add, subtract, multiply, and divide two numbers.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## How to run it
1. Compile the code:
   g++ calculator.cpp -o calculator
2. Run the program:
   ./calculator

## Example
Enter first number: 5
Enter second number: 3
Choose operation (+, -, *, /): +
Result: 8

This is one of my first programming projects. I made it to practice C++ basics.
